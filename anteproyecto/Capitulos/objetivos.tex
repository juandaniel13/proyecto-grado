\chapter{Objetivos}

\section{Objetivo General}

Desarrollar una aplicación web con IA para la simulación y análisis
de arquitecturas LPWAN en proyectos de agricultura inteligente en zonas rurales de Colombia.

\section{Objetivos Específicos}

\begin{enumerate}
      \item Determinar los requerimientos funcionales y no funcionales de la
            aplicación web para definir el alcance y el comportamiento
            esperado de la aplicación.
      \item Diseñar una arquitectura de software que integre agentes de
            inteligencia artificial y que responda a los requerimientos
            funcionales y no funcionales definidos para la aplicación.
      \item Implementar la aplicación web mediante un esquema de desarrollo
            ágil basado en la metodología XP.
      \item Evaluar la calidad funcional de la aplicación web mediante
            la ejecución de pruebas de aceptación sobre los requisitos
            funcionales definidos
      \item Documentar la arquitectura de software y el proceso de
            desarrollo para garantizar la
            divulgación de resultados de investigación.
\end{enumerate}
